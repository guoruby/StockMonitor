## 📖 使用步骤

### 第一步：配置监控区域（首次使用）

1. **点击"设置区域"按钮**
   - 会弹出一个配置窗口

2. **点击"开始选择"按钮**
   - 程序会捕获整个屏幕
   - 顶部显示黄色文字："点击[最新价]位置"

3. **点击同花顺的【最新价】**
   - 用鼠标点击同花顺软件中最新价数字的位置
   - 程序会用**绿色框**标记该位置

4. **点击同花顺的【均价】**
   - 顶部文字变为："点击[均价]位置"
   - 用鼠标点击同花顺软件中均价数字的位置
   - 程序会用**蓝色框**标记该位置

5. **完成选择**
   - 顶部文字变为："设置完成！点击任意处关闭"
   - 点击屏幕任意位置关闭截图窗口
   - 返回配置窗口，可以看到选择的坐标

6. **保存配置**
   - 点击"保存"按钮
   - 配置会保存到 `monitor_config.json` 文件
   - 主窗口底部显示："配置已保存！可以开始监控了"

### 第二步：开始监控

1. **点击"开始监控"按钮**
   - 按钮变为红色"停止监控"
   - 底部显示："监控中..."

2. **观察数据显示**
   - **最新价**: 绿色=上涨，红色=下跌
   - **均价**: 白色显示
   - **偏离**: 显示偏离值和百分比
   - **更新时间**: 最后更新时间

3. **偏离提醒**
   - 当偏离百分比 ≥ 5% 时：
     - 数字变为醒目的红色或紫色
     - 显示"上"或"下"箭头
     - 底部说明提醒需要注意

4. **停止监控**
   - 点击"停止监控"按钮
   - 按钮变回绿色"开始监控"
   - 底部显示："监控已停止"

### 第三步：调整窗口

- **拖动窗口**: 用鼠标点击窗口任意位置并拖动
- **窗口始终置顶**: 不会被其他窗口遮挡
- **深色主题**: 适合长时间盯盘

## ⚠️ 重要提示

### 首次使用
- 现在使用的是**模拟数据**（10-50之间的随机数）
- 配置真实监控区域后，会使用OCR识别真实价格

### 配置监控区域
- 确保同花顺软件在前台
- 不要被其他窗口遮挡
- 点击数字的**中心位置**
- 如果同花顺窗口移动，需要重新设置

### OCR识别准确度
- 使用高对比度主题
- 使用较大字体（≥12pt）
- 监控区域只包含数字，不要包含多余空白
- 避免使用半透明或模糊显示

## 🔧 配置文件

配置文件位置：`monitor_config.json`

示例内容：
```json
{
  "current_price_region": {
    "top": 200,
    "left": 800,
    "width": 100,
    "height": 30
  },
  "avg_price_region": {
    "top": 240,
    "left": 800,
    "width": 100,
    "height": 30
  }
}
```

### 参数说明
- `top`: 区域上边距（像素）
- `left`: 区域左边距（像素）
- `width`: 区域宽度（像素）
- `height`: 区域高度（像素）

### 手动编辑
如果需要微调，可以直接编辑 `monitor_config.json` 文件。

## 🐛 常见问题

### Q: 配置窗口打不开？
A: 确保程序正在运行，点击"设置区域"按钮

### Q: 截图窗口不显示？
A: 检查同花顺窗口是否在前台，尝试重新点击"开始选择"

### Q: OCR识别不准确？
A:
1. 重新设置监控区域，只包含数字
2. 调整区域大小（100x30像素）
3. 使用高对比度主题
4. 增大字体大小

### Q: 偏离显示为0或NaN？
A:
1. 检查监控区域是否正确
2. 确认同花顺软件正常运行
3. 检查OCR是否识别到数字（查看终端输出）

### Q: 程序退出后配置丢失？
A: 配保存在 `monitor_config.json`，重新启动会自动加载

### Q: 如何修改偏离阈值？
A: 编辑代码中的条件（第267行左右）：
```python
if abs(self.deviation_percent) >= 5:  # 修改这个数值
```
