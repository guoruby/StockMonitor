# 股票价格监控程序

## 🚀 快速开始

```bash
# 方式1：使用启动脚本
./start.sh

# 方式2：直接运行
python3 stock_monitor_final.py
```

## 📖 使用步骤

1. **配置监控区域**
   - 点击"设置区域" → "开始选择"
   - 点击同花顺的【最新价】
   - 点击同花顺的【均价】
   - 点击"保存"

2. **开始监控**
   - 点击"开始监控"
   - 观察价格偏离
   - 偏离≥5%时红色提醒

3. **调整窗口**
   - 拖动到合适位置

## ⚙️ 系统要求

- macOS 10.13+
- Python 3.7+
- Tesseract OCR

## 💡 提示

- 首次使用需要配置监控区域
- 配置会保存到 `monitor_config.json`
- 同花顺窗口移动后需要重新配置
